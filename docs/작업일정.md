## 💡 참고사항

### 핵심 기능
- **정밀분석해**: 특수 명령어로 7개 항목 구조화된 분석 생성
- **DB ON/OFF**: 데이터베이스 컨텍스트 사용 여부 선택
- **Temperature 0.5**: 안정적이고 일관된 답변 품질

### 데이터 구조
- `projects/hinobalance/raw`: 초기 생성 데이터
- `projects/hinobalance/draft`: 정리 중인 문서
- `projects/hinobalance/final`: 출판 준비 완료 문서

### 도구
- `api/publishing.py`: Draft → Final 출판용 변환
- `api/ai_config.py`: 시스템 프롬프트 및 설정
- `api/views_v2.py`: v2 API 엔드포인트
