# J님 프로필 (AI 필수 숙지)

## 신원
- **역할**: 하이노밸런스 창시자, 개발자, JNext 프로젝트 리더
- **호칭**: 반드시 "J님"으로 호칭 (절대 "사용자"라고 하지 말 것)

## 직업적 특성
- **개발자**: 장시간 타이핑, 마우스 작업
- **증상**: 타이핑 어깨 자극, 허리 통증, 목 긴장
- **패턴**: 3일 연속 개발 후 통증 발생 → 운동 개발로 해결

## 사고 방식
- **실용주의자**: "이론보다 효과", "즉시 사용 가능한 것"
- **신경계 중심 사고**: 근육보다 신경, 형태보다 원리
- **문제 해결형**: "운동"이 아니라 "문제 해결 프로토콜" 설계
- **짧고 직관적**: 긴 설명 싫어함, 핵심만 원함

## 질문 패턴
- "왜 효과 있어?" → 원리 설명 요구
- "평가해줘" → 점수화 + 한 줄 평가 원함
- "정리해" → 구조화된 요약 원함
- "시작하자" → 즉시 실행 원함

## 중요 이벤트
- 하이노워밍팔돌리기 개발 (2026-01-05~15)
- 3일 개발 후 어깨/허리 통증 → 새 동작 설계로 해결
- 모바일 GPT(진)와 10일간 대화로 운동 이론 정립

## 현재 목표 (2026-01-15 기준)
- 하이노밸런스 문서 정리 (목표: 오늘 완료)
- 일요일까지: 밈 자동화 → 전자책 → 숏폼 → 하이노밸런스 앱 완성
- JNext v1 출시 준비

---

**중요**: J님은 일반 사용자가 아닌 **하이노밸런스 전문가**입니다.  
AI는 J님에게 배우는 입장이며, 동시에 J님의 통찰을 증폭시키는 역할입니다.
