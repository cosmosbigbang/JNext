# 하이노밸런스 핵심철학 (J님 원본)

## 슬로건
- 안녕 불균형, 반가워 불균형
- 정지 = 균형 = 죽음
- 한발 = 출발 = 삶

## 핵심 원리

### 1. 한발 운동 체계
- **하이노(Hino)** = 형태적으로 "한발"이란 뜻
- **두발 = 정지 = 죽음**
- **한발 = 출발 = 삶**
- 모든 운동이 한발 운동으로 세팅됨

### 2. 불균형의 철학
- 한발로 균형을 이루려 애쓰지만... 균형을 이루면 일부러 미세 불균형 일으킴
- 균형이 깨지면 발을 자연스럽게 바꾸며 재시작
- **균형은 지나치는 것**
- 억지로 균형을 이루려고 하지 않음
- 노력하다 깨지면 발 바꿔 다시 시작

### 3. 가속도 원리
- 동작을 똑같은 속도로 반복하는 것은 초기 단계
- 몸이 풀리면 **정지 동작을 삽입**
- 움직이다 정지하면 → 이것은 정지가 아닌 **가속도**
- **가속도 통제로 뇌를 자극**
- **가속도로 뇌를 깨우는 운동**

### 4. 신경 중심 운동
- 나이 들면 근육과 신경이 모두 퇴화
- 보통의 운동 = 근육 중심
- **하이노밸런스 = 신경 자극으로 신경을 강화**
- 더불어 근력과 중심감을 증가시킴

## 목표
- 가속도로 신경을 자극
- 한발 운동으로 불균형 통제
- 근력, 심폐, 신경계 모두 향상
