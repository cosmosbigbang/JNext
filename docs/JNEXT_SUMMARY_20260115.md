# JNext 프로젝트 요약 (2026.01.15)

## 🎯 프로젝트 목표
AI 기반 콘텐츠 자동화 시스템 - 문서 관리부터 밈, 숏폼 영상까지 완전 자동화

---

## 🏗️ 현재 완성된 시스템

### 1. **3단계 문서 관리 시스템**
- **RAW** → **DRAFT** → **FINAL** 워크플로우
- Firestore 기반 문서 DB
- 프로젝트별 컬렉션 관리 (hinobalance 등)

### 2. **AI 재생성 피드백 루프** ⭐ (핵심)
```
재생성 클릭 → AI가 30개 문서 맥락 분석 → 새 내용 생성
↓
기존 vs 새 내용 비교 + 직접 수정
↓
피드백 입력 (선택)
↓
AI가 피드백 반영하여 개선
↓
최종 적용
```

**특징:**
- 프로젝트 전체 맥락 이해 (30개 문서 동시 분석)
- 즉시 수정 가능 (textarea)
- 마크다운 자동 제거
- 정확한 명령 이행 (temperature 0.3)

### 3. **웹 인터페이스**
- **채팅 페이지**: 동적 컨텍스트 기반 대화
- **문서 관리 페이지**: 검색, 필터, 정렬, 수정, 삭제, 재생성
- 페이지 간 네비게이션 버튼

---

## 🔧 기술 스택
- **Backend**: Django 6.0, Python 3.14
- **Database**: Firestore
- **AI**: Gemini Pro (재생성 0.85, 피드백 0.3)
- **Frontend**: Vanilla JS, HTML/CSS
